PASO A PASO:

1. Ejecutar el programa "ProgramaMatriz.py" a traves de la terminal
2. Observar la Matriz disponible y el numero de casillas que quedan por seleccionar. (5 si es primera vez)
3. Ingresar la fila donde se quiere añadir el elemento (de 1 a 10)
4. Ingresar la columnda donde se quiere añadir el elemento (de 1 a 5)
5. Repetir el proceso seleccionando las casillas validas.
6. Observar la matriz finalizada.

** En caso de seleccionar una casilla no valida, el programa avisara al usuario de que la casilla no es valida.

IMPORTANTE: La ejecucion de este programa debe hacerse con el lenguaje de programacion python.
