PASO A PASO: 

1. Ejecutar el programa "ProgramaMorse.py" a traves de la terminal
2. Escoger una opcion del menu:
	"Pulse 1 Si desea pasar de Morse a Alfabeto"
	"Pulse 2 Si desea pasar de Alfabeto a Morse"
3. En caso de escoger 1, se debe ingresar una cadena en codigo morse.
4. En caso de escoger 2, se debe ingresar una cadena en lenguaje natural.
5. El resultado final se imprimira por pantalla y terminara la ejecucion del programa.

IMPORTANTE: La ejecucion de este programa debe hacerse con el lenguaje de programacion python.
