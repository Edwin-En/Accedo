PASO A PASO:

IMPORTANTE: Para la ejecucion completa de este programa es necesario contar con la herramienta XAMP activando el modulo Apache y MySQL.
Tambien es importante verificar la creacion de la database dentro del phpMyAdmin esta se llama: "pokemonapi_database"


1. Iniciar la herramienta XAMP.
2. En el navegador de internet de preferencia buscar la URL:
localhost/PokemonAPI
3. Registrase (para ello se requiere un correo, contraseña y su confirmacion).
4. Una vez registrado se puede iniciar sesion, con el correo y la contraseña registrados.
5. Actualmente la pantalla del index muestra un mensaje de bienvenida y da dos opciones, terminar la sesion o ver la lista de pokemones.
6. Si se deseea ver la lista de pokemones disponibles, se da clic en el ultimo enlace.
7. Se carga una lista con los pokemones de la pagina, si se desea conocer a detalle cada pokemon, se debe dar clic.
8. Se observan los datos del pokemon como nombre, habilidades y fotos.

Nota: La aplicacion web cuenta con un menu de navegacion el cual al pulsar "Pokemon App" redirige al usuario al index.php
